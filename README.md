## 特徴
暗号化やハッシュを求める、CTFなどのイベントなどに使える便利なツールです。 
シーザー暗号などの古典暗号なども対応しています。 

## 独自実装および対応暗号・ハッシュ・フォーマット
### 暗号
シーザー暗号、AES暗号 等
### ハッシュ
 MD5・SHA2・HMAC等
 ### フォーマット
 Base64 等
